# Glipped

A simple Express App to invite users to Glip Groups.
 
 
## Requirement

* Node - 7.4.0  
* Express - 4.11.xx 


## Installation via NPM

```bash
$ cd glipped
$ npm install
```

## Run the Program


```bash
$ cp config-sample.env.txt .env
$ vi .env
$ npm start
```


Go to the URL:

```
http://localhost:8080
````

Then click the `Glip` badge to invite users to Glip.
