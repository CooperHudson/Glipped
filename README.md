# Glipped

A quick way to imple Express App to invite users to Glip Groups.
 
 
## Requirement

Node.js 4.2 as mimimum.  
Express 4.11.xx as minimum.



## Installation via NPM

```bash
$ cd glipped
$ npm install
```

## Run the Program


```bash
$ cp config-sample.env.txt .env
$ vi .env
$ npm start
```


Go to the URL:

```
http://localhost:8080
````

Then click the `Invite` button to invite users to Glip.
